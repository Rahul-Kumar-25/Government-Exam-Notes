What is Syllogism? (Level 0)

Syllogism is a type of logical reasoning question. In this topic, you are given two or three statements and based on those statements, you have to decide whether a conclusion is correct or not.

Important rule: You are not allowed to use real-world knowledge. Only believe what is written in the statements.

Example

Statements:

  • All apples are fruits.
  • All fruits are sweet.

Conclusion: All apples are sweet. (True)

Basic Building Blocks of Syllogism

Word Meaning
All 100%
Some At least one
No Zero
Some…not At least one is outside

Four Basic Types of Statements

1. All A are B

Meaning: Every A is inside B.

Example: All dogs are animals.

2. Some A are B

Meaning: At least one A overlaps with B.

Example: Some students are boys.

3. No A is B

Meaning: A and B never overlap.

Example: No cat is a dog.

4. Some A are not B

Meaning: At least one A is outside B.

Example: Some boys are not tall.

Most Important Rule

Never assume extra information. Only what is definitely stated or guaranteed is allowed.

Example

Statement: Some boys are smart.

  • Some boys are not smart – Not correct
  • All boys are smart – Not correct

How Questions Are Asked in Exams

Statements:

  • All A are B
  • Some B are C

Conclusions:

  1. Some A are C
  2. Some C are A

Method 1: Venn Diagram Method

This is the best method for beginners. Draw circles to represent the statements and check whether the conclusion is definitely true.

Example

Statements:

  • All A are B
  • All B are C

Conclusion: All A are C (True)

Method 2: Logical Thinking Method

After practice, you can solve questions without drawing diagrams by applying logical rules.

Rule 1: All to Some Conversion (Valid)

All A are B → Some A are B

Rule 2: Some to All (Invalid)

Some A are B → All A are B (Wrong)

Rule 3: No Direct Negative Conclusion

From "All A are B", you cannot conclude "Some A are not B".

Conversion Rules

Valid Conversions

  • All A are B → Some B are A
  • No A is B → No B is A
  • Some A are B → Some B are A

Invalid Conversions

  • Some A are not B → Some B are not A (Wrong)

Bank PO / Clerk Level Concepts

1. Possibility Case

If a conclusion may or may not happen but is not guaranteed, it is called a possibility case.

Example

Statements:

  • All A are B
  • Some B are C

Conclusion: Some A are C (Possibility)

2. Either – Or Case

If two conclusions contradict each other, only one can be true.

Example

  • Some A are B
  • Some A are not B

Answer: Either I or II

3. Definitely True vs Possibly True

Type Meaning
Definitely True 100% follows
Possibly True Can happen but not guaranteed

Common Mistakes

  • Using real-life knowledge
  • Assuming extra information
  • Confusing some with all
  • Wrong conversion of some-not statements

Final Exam Strategy

  1. Use Venn diagrams initially
  2. Check only definite conclusions
  3. Apply conversion rules carefully
  4. Check either-or cases
  5. Handle possibility cases carefully

Golden Line

In syllogism, do not think like a human. Think like a machine.
